# react-native-hermes-worker

A react-native js woker using hermes runtime

## Installation

```sh
npm install react-native-hermes-worker
```

## Usage


```js
import { multiply } from 'react-native-hermes-worker';

// ...

const result = multiply(3, 7);
```
